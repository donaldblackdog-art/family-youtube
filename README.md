# 가족용 유튜브 MVP

오래된 노트북이나 NAS처럼 집에 있는 장비에서 가족 영상 목록과 재생 페이지를 제공하는 작은 MVP입니다.

## 지금은 어디서 작업하면 좋나요?

개발은 지금 쓰시는 맥북에서 하시는 편이 좋습니다. 화면 확인, 코드 수정, 테스트가 빠릅니다.

완성 후에는 오래된 노트북으로 이 폴더를 옮기고 그 노트북에서 `npm start`를 실행하면 됩니다. 외부 가족이 언제든 접속하려면 그 노트북은 전원과 인터넷이 켜져 있어야 합니다.

## 실행

```bash
npm start
```

브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:3000
```

기본 비밀번호는 `0000`입니다.

비밀번호를 바꾸려면 실행할 때 환경변수를 넣습니다.

```bash
FAMILY_PASSWORD="원하는비밀번호" npm start
```

## 영상 저장 위치

업로드한 영상은 아래 폴더에 저장됩니다.

```text
media/videos
```

썸네일은 아래 폴더에 저장됩니다.

```text
media/thumbnails
```

영상 목록 데이터는 아래 파일에 저장됩니다.

```text
data/videos.json
```

## 썸네일 자동 추출

컴퓨터에 `ffmpeg`가 설치되어 있으면 업로드한 영상의 2초 지점에서 썸네일을 자동 생성합니다.

`ffmpeg`가 없으면 기본 썸네일을 사용합니다.

## 외부 가족에게 보여주기

외부 접속용으로는 Cloudflare Tunnel을 추천합니다.

```text
가족의 브라우저 -> Cloudflare Tunnel 주소 -> 오래된 노트북의 http://localhost:3000
```

이 MVP는 가족 비밀번호로 한 번 막아두는 구조입니다. 나중에 영상별 비밀 링크, 가족별 계정, 다운로드 차단, 댓글 기능을 추가할 수 있습니다.
