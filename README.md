# 초초TV MVP

구글드라이브에 올려둔 가족 영상 링크를 모아 가족끼리 볼 수 있게 해주는 작은 MVP입니다.

## 지금은 어디서 작업하면 좋나요?

개발은 지금 쓰시는 맥북에서 하시는 편이 좋습니다. 화면 확인, 코드 수정, 테스트가 빠릅니다.

완성 후에는 오래된 노트북으로 이 폴더를 옮기고 그 노트북에서 `npm start`를 실행하면 됩니다. 외부 가족이 언제든 접속하려면 그 노트북은 전원과 인터넷이 켜져 있어야 합니다.

## 실행

```bash
npm start
```

브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:3000
```

기본 가족 보기 비밀번호는 `0000`입니다.

관리자 로그인 주소는 아래와 같습니다.

```text
http://localhost:3000/admin
```

기본 관리자 비밀번호는 `9999`입니다. 관리자로 들어온 사람만 구글드라이브 영상 링크 등록, 수정, 삭제를 할 수 있습니다.

비밀번호를 바꾸려면 실행할 때 환경변수를 넣습니다.

```bash
FAMILY_PASSWORD="가족비밀번호" ADMIN_PASSWORD="관리자비밀번호" npm start
```

## 영상 저장 방식

영상 원본은 초초TV 서버에 저장하지 않습니다. 구글드라이브에 직접 올린 뒤, 초초TV 관리자 화면에 공유 링크만 등록합니다.

```text
구글드라이브 영상 파일 -> 초초TV에 공유 링크 등록 -> 가족 화면에서 재생
```

영상 목록 데이터는 아래 파일에 저장됩니다.

```text
data/videos.json
```

구글드라이브 영상은 공유 권한이 가족들이 볼 수 있게 설정되어 있어야 합니다. 보통 영상 파일이나 폴더를 `링크가 있는 모든 사용자 - 뷰어`로 설정합니다.

## 구글드라이브 링크 등록

1. 구글드라이브에 영상을 올립니다.
2. 영상 파일을 우클릭하고 공유 링크를 복사합니다.
3. 초초TV에 관리자로 로그인합니다.
4. `Drive 링크 등록`을 눌러 제목, 촬영 날짜, 공유 링크를 입력합니다.
5. 등록된 영상은 가족 화면에서 바로 볼 수 있습니다.

## 삭제

초초TV에서 삭제하면 목록에서만 사라집니다. 구글드라이브에 있는 원본 영상은 삭제되지 않습니다.

## 외부 가족에게 보여주기

외부 접속용으로는 Cloudflare Tunnel을 추천합니다.

```text
가족의 브라우저 -> Cloudflare Tunnel 주소 -> 오래된 노트북의 http://localhost:3000
```

이 MVP는 가족 비밀번호로 한 번 막아두는 구조입니다. 나중에 영상별 비밀 링크, 가족별 계정, 다운로드 차단, 댓글 기능을 추가할 수 있습니다.
