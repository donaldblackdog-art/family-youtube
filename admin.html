How to add

구글드라이브 영상 추가 방법

이 사이트는 서버가 없어서 사이트 안에서 바로 저장하지 않습니다. 대신 `videos.js` 파일에 구글드라이브 링크 정보를 추가하고 GitHub에 올리면 됩니다.

1

구글드라이브에 영상 올리기

구글드라이브에 `초초TV` 폴더를 만들고 영상을 올립니다.

2

공유 링크 복사

영상 파일 공유 권한을 `링크가 있는 모든 사용자 - 뷰어`로 바꾼 뒤 링크를 복사합니다.

3

영상 목록에 추가

복사한 링크를 `videos.js`의 `driveUrl`에 넣고 제목, 촬영일, 설명을 적습니다.

추가할 영상 예시

{
  title: "제주 가족여행",
  recordedDate: "2026-07-21",
  addedDate: "2026-07-21",
  description: "할머니, 할아버지와 함께 본 바다",
  driveUrl: "https://drive.google.com/file/d/구글드라이브파일ID/view"
}
